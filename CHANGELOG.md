# Changelog

## v1.1.3

EN:

- Alcohol-level based rules (`alcoholRules`) for chat replacement (chance + random phrases per range).
- Optional Carbon Chat integration (reflection + `carbonchat` entrypoint).
- Optional permissions integration (Permissions API / LuckPerms) for `/brreload` and bypass permission.
- Build artifacts are published to `build/libs/release_<version>/`.

UA:

- Правила по рівню алкоголю (`alcoholRules`) для підміни чату (шанс + випадкові фрази для діапазонів).
- Опціональна інтеграція з Carbon Chat (reflection + entrypoint `carbonchat`).
- Опціональні права (Permissions API / LuckPerms) для `/brreload` та bypass-права.
- Артефакти збірки публікуються в `build/libs/release_<version>/`.
